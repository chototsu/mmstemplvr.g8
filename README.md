mmstemplvr.g8
=============

# VR Template for MikuMikuStudio

## Original
http://hub.jmonkeyengine.org/forum/topic/oculus-rift-support/

## Usage
Create a project:
g8 chototsu/mmstemplvr

Run on PC(Windows, MacOS X, Linux)

cd (project folder)
sbt
run

Run on Android
cd (project folder)
sbt
project android-project
run
